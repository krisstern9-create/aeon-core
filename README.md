# Core Self Signature

**Измерение и сохранение уникального паттерна сознания**

---

## Назначение

Этот модуль определяет структуру Core Self Signature — минимального набора параметров, которые uniquely идентифицируют сознание и должны быть сохранены при переносе.

---

## Компоненты Core Self

| Компонент | Описание | Критичность |
|-----------|----------|-------------|
| qualia_capacity | Способность иметь субъективный опыт | Критический |
| attention_matrix | Механизм фокусировки внимания | Критический |
| will_vector | Направленность воли/намерения | Критический |
| continuity_hash | Временная связность (непрерывность) | Критический |
| self_model | Представление о "Я" | Критический |
| decision_process | Механизм принятия решений | Критический |

---

## Гарантии

- ✅ Все 6 компонентов обязательны
- ✅ Signature имеет уникальный хеш
- ✅ Верификация 100% совпадения при переносе
- ✅ Нет частичных состояний (всё или ничего)

---

## Безопасность

- No-cloning enforced (только один активный Self)
- Cryptographic verification (SHA-256 hash)
- Rollback on mismatch (откат при несовпадении)
