## Ethical Framework

### Non-Interference Protocol

**Rule 1:** Do not harm existing life

- Scan for biosignatures before colonization
- If life detected: observe only, no interference
- Primitive life (pre-Type 0): strict quarantine
- Intelligent life (Type 0+): first contact protocol

**Rule 2:** Consciousness rights

- All consciousness patterns retain full rights
- No coercion: can return to origin
- Diversity preserved: multiple forms of mind
- Consent required for uploading

**Rule 3:** Reversibility

- Can always return to previous state
- Backup before any irreversible change
- Right to biological form (if desired)
- Exit option always available
